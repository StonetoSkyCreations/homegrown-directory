# Archive

Point-in-time reports kept for history. They were snapshots of the site at a
moment in time, not living documents, so they are parked here and excluded from
the build.

The living sources of truth are, in the repo root:

- `README.md` — architecture, how to add a listing, the listing-quality standard.
- `ROADMAP.md` — what is done, known issues, what is next.
- `SEO_DOCTRINE.md` — the canonical SEO rules enforced by `scripts/seo_lint.py`.

Anything in these archived files that is still true has been folded into those
three. Do not treat the files below as current.
